flask.debughelpers.FormDataRoutingRedirect

FormDataRoutingRedirect: A request was sent to this URL (http://localhost:8084/classify) but a redirect was issued automatically by the routing system to "http://localhost:8084/classify/". The URL was defined with a trailing slash so Flask will automatically redirect to the URL with the trailing slash if it was accessed without one. Make sure to directly send your POST-request to this URL since we can't make browsers or HTTP clients redirect with form data reliably or without user interaction. Note: this exception is only raised in debug mode
